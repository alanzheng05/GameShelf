# GameShelf Frontend

This project frontend is built with Angular 21.

## Prerequisites

- Node.js (LTS recommended)
- npm

## Install

```bash
cd frontend
npm install
```

## Run Frontend

```bash
cd frontend
npm start
```

Open: `http://localhost:4200/`

## Other Frontend Commands

```bash
cd frontend
npm run build
npm test
```

## Notes

- Unit tests run with `ng test` (Vitest setup).
- No e2e target is currently configured.
